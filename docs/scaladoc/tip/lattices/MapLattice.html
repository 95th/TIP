Related Doc: package lattices

class MapLattice[A, +L <: Lattice] extends Lattice

A lattice of maps from the set X to the lattice l. The set X a subset of A and it is defined by the characteristic function ch, i.e. a is in X if and only if ch(a) returns true. Bottom is the default value.
