TypeAnalysis

Related Doc: package analysis

class TypeAnalysis extends DepthFirstAstVisitor[Null] with Analysis[TypeData]

Unification-based type analysis. The analysis associates a tip.types.TipType with each variable declaration and expression node in the AST. It is implemented using tip.solvers.UnionFindSolver.
